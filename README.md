# Chicago-Traffic-Data

Chicago Traffic Data program shows data about red light and speed cameras in Chicago. Data includes violations, number of cameras, etc. 

This program was developed in Python.

# Features

- 9 menu options showing different statistics
- Line graph of data for some options
- Wide collection of data from traffic database
